# Проблема
## Краткое описание
Когда TypeScript (npm) библиотека вырастает до больших размеров, контролировать используемый код становится очень сложно.
При экспорте объектов из файла неочевидно, будут ли они использоваться в публичном API или нет.
Если экспортируемые сущности не используются внутри библиотеки, они могут долго оставаться в коде как мёртвый груз.

## Целевая аудитория
Программисты, которые поддерживают большие библиотеки, хотят быстро находить мёртвый код и удалять его.

## Ценностное предложение
Упрощает поддержку больших TypeScript (npm) библиотек, позволяя быстро идентифицировать и удалять мёртвый код.

## Основные характеристики
- Автоматическое обнаружение мёртвого кода
- Интеграция с TypeScript (npm) проектами
- Простота использования и настройки
- Поддержка различных типов мёртвого кода
